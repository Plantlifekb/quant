Quant v1.0 — Factor Exposure Specification

Canonical, governed, non‑drifting

Purpose

Define the exact factor definitions, exposure calculations, normalization rules, and invariants used in Quant v1.0. This ensures that all factor‑related metrics are deterministic, auditable, and free from hidden transformations or external data.

1. Factor Philosophy

Quant v1.0 uses a minimal, deterministic, price‑derived factor framework. It must:

rely only on governed inputs

avoid external factor datasets

avoid probabilistic or model‑based factors

avoid adaptive or regime‑dependent factor weights

remain fully explainable and reproducible

Factors measure structural characteristics, not predictions.

3. Factor Definitions

Quant v1.0 uses three governed factors, all derived from price.

3.1 Momentum Factor (MOM)

Definition: 20‑day return

[ MOM(t) = ret_{20d}(t) ]

Interpretation:

Positive = upward trend

Negative = downward trend

3.2 Volatility Factor (VOL)

Definition: 20‑day population standard deviation of daily returns

[ VOL(t) = vol_{20d}(t) ]

Interpretation:

Higher = more unstable

Lower = more stable

3.3 Short‑Term Return Factor (STR)

Definition: 1‑day return

[ STR(t) = ret_{1d}(t) ]

Interpretation:

Captures short‑term price pressure

4. Factor Normalization

4.1 Cross‑Sectional Z‑Scores

All factors are normalized daily using:

[ z(x_i) = \frac{x_i - \mu}{\sigma} ]

Where:

(\mu) = cross‑sectional mean

(\sigma) = cross‑sectional standard deviation

4.2 No Alternative Normalizations

Forbidden:

min‑max scaling

rank‑based scaling

winsorization

clipping

volatility‑adjusted scaling

5. Factor Exposure Calculation

5.1 Exposure Vector

Each ticker has a factor exposure vector:

[ F_i = [ MOM_i, VOL_i, STR_i ] ]

5.2 Normalized Exposure Vector

After z‑scoring:

[ Z_i = [ z(MOM_i), z(VOL_i), z(STR_i) ] ]

5.3 No Factor Weighting

Quant v1.0 does not compute a weighted factor score.

Factor exposures are informational only.

6. Factor Invariants

No external factor datasets

No macroeconomic factors

No fundamental factors

No alternative data

No regime‑dependent factor weights

No dynamic factor definitions

No smoothing or filtering

No randomness
